日常爬坑记录

发表于 2019-07-12 | 分类于 IDE

这篇博客用来记录平常遇到的坑。

idea文件导入

从github克隆到本地的代码，在用idea打开时，在拉取下来后，不要用import导入，选择open打开。

Ubuntu进不了桌面

最近在ubuntu上装环境时跳出来说有部分软件需要更新，于是我点了更新，但是提示我更新失败。之后再打开虚拟机时发现进不去ubuntu的桌面了，只显示ubuntu的图标，没有登录密码的输入框，即卡在登录页面之前。找了好久问题，后来在这篇贴吧上找到了跟我同样问题的人。之前其实根据这篇经验是可以进入到桌面的，但是它里面说的解决方案对我不适用。
其实之前我的ubuntu18.04虚拟机默认使用的是一个叫做gdm3的桌面管理环境，估计在更新的时候出现了问题。于是根据那篇贴吧，我使用lightdm作为我的桌面管理环境，在终端输入sudo apt-get install lightdm即可完成lightdm的下载安装，在安装过程中它会让你选择默认的桌面管理环境，只需选择lightdm就行了。
